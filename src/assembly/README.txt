This distribution contains binaries to run grand challenge problem as a standalone program. 

To run both queries use 
sh gc.sh <location of file>  eg.  sh gc.sh ../../data/sorted_data.csv

To run only route query
sh route.sh <location of file>  eg.  sh route.sh ../../data/sorted_data.csv

To run only profit query
sh profit.sh <location of file>  eg.  sh profit.sh ../../data/sorted_data.csv
